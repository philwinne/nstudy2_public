You can access SPINStudy by following these steps:
  1. Download latest version of Chrome web browser

  2. Open the Chrome web browser.

  3. In the address field, enter this URL: https://chrome.google.com/webstore/detail/nstudy/mmmiodfchlgjfnbnpgaecgnmhbdpceki

  4. This page will appear. Click on "ADD TO CHROME"



  5. A box will open, click "Add extension"


  6. Then this message will appear



  7. nStudy icon will appear on the top right of the chrome address bar. It is all black when inactive



  8. To create an nStudy account, go to: https://nstudy.educ.sfu.ca/ns4/user/login/

  9. Click "Create New Account"


  10. Enter your email and password, then click Submit
