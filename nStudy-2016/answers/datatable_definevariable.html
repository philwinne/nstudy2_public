1. Double click on the name. A window will open with options


  2. Click in the Column name field to type the name of the variable (column)



  3. Click in in the Units field to type unit for the variable.



  4. You can add notes by clicking on the Notes field and entering information.
  5. Click on Types to open a menu with options to identify data in this column as (Number, Text, or Formula).
  6. Click on Scale to open a menu with options to describe the scale of the data for this variable (Continuous, Nominal, Ordinal).
  7. If you want to create a column that is made up of other columns i.e. using a formula (e.g. Sum, Division...etc.), click on the formula option Select variables using variable picker on left side and clicking right-caret button (>) - Support for +, -, *, /, ^ operators.
  8. (If there are no errors), pressing Ok will apply the formula to all rows (cases) in the column.
